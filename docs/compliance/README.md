## Placeholders

All documents contain `[PLACEHOLDER]` markers (in square brackets) where company-specific information must be filled in before publication. Common placeholders include:

- `[COMPANY NAME]` -- Legal name of the company operating ConsolidaSuite
- `[COMPANY ADDRESS]` -- Registered business address
- `[DPO NAME]` / `[DPO EMAIL ADDRESS]` -- Data Protection Officer details
- `[JURISDICTION]` -- Governing law jurisdiction
- `[SUPERVISORY AUTHORITY NAME]` -- Relevant data protection authority

Search for text in square brackets to find all placeholders that need to be completed.

---

## Review Schedule

All documents should be reviewed:

- **Annually** as part of the regular compliance review cycle.
- **When material changes** occur to the platform, data processing activities, or sub-processors.
- **After any data breach** or significant security incident.

---

## Applicable Regulations

These documents are designed to support compliance with:

- **GDPR** (General Data Protection Regulation, EU 2016/679)
- **ePrivacy Directive** (Directive 2002/58/EC) -- cookie requirements
- **National implementations** of the GDPR in relevant EU Member States
